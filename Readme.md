## Basic Structure
```
src/
 ├─ models/        # Mongoose 
 ├─ controllers/   # HTTP request handlers
 ├─ services/      # Business logic
 ├─ routes/        # Express routes
 ├─ middleware/    # Auth, Validation, Rate-limit
 └─ index.ts
 ```
